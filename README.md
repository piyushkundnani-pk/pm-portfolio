## Module 1 — Amazon Smart Shopping Companion

**Type:** Product Concept | **Domain:** E-Commerce | **Platform:** Amazon India

### Problem
Amazon's catalog-first model creates decision fatigue for 25–30M guidance-seeking users in India — elders, rural/semi-urban shoppers, and occasion-driven buyers who need a salesperson, not a search bar.

### Solution
**Smart Shopping Companion** — an AI-guided assistant that asks 3 contextual questions (occasion, mood, budget) and returns curated product recommendations, replicating the in-store concierge experience.

### Key Artifacts
- ✅ Problem Statement & JTBD
- ✅ 3 User Personas (Meena Devi, Rakesh Patel, Shruti Jain)
- ✅ Customer Journey Map (Rakesh Patel)
- ✅ TAM/SAM/SOM Opportunity Sizing (~₹25 Cr Year 1)
- ✅ MoSCoW Prioritization
- ✅ MVP Feature Set + Wireframes
- ✅ Now/Next/Later Roadmap
- ✅ AI Component Strategy (NLP, Recommendation Engine, Sentiment Analysis)
- ✅ Metrics Hierarchy (NSM → L1 → L2)

### North Star Metric
**Guided Shopping Conversion Rate** — % of users completing a purchase after using the Smart Shopping Companion
